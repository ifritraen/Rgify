description: Perform a deep audit and polish of the entire project to fix bugs and elevate quality to highest standards.
---
When the user executes `/overhaul` or starts their request with `/overhaul`:
1. Run the appropriate static analysis or lint tool based on the project type (e.g., `flutter analyze` for Flutter, `npm run lint` for JS/TS, etc.) to locate and resolve any compiler warnings, type errors, or deprecations.
2. Review all core application features and UI/UX screens for premium aesthetics, responsiveness, seamless user flows, and robust caching/data persistence.
3. Check and polish styling components, color consistency, typography, scroll indicators, and animations to ensure a premium, modern visual experience.
4. Correct any discovered bugs, warnings, or structural defects permanently, tracing issues to their absolute root causes.
